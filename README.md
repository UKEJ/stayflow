# StayFlow

StayFlow is a modern, cloud-based Hotel Management System (HMS) and Booking Platform built to help hotels digitize their operations.

## Features

- Hotel Website Builder
- Online Booking Engine
- Property Management System (PMS)
- Room Management
- Staff & Role Management
- Housekeeping
- Maintenance Tracking
- Restaurant & POS
- Analytics Dashboard
- Customer Relationship Management
- AI-powered Business Insights
- Payment Integration
- Multi-property Management

## Tech Stack

Frontend
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

Backend
- Laravel
- PostgreSQL
- Redis

Infrastructure
- Docker
- Cloudflare
- GitHub Actions

## Vision

To become the operating system that powers independent hotels across Africa.
